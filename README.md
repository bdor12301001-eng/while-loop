حلقة while تُستخدم لتكرار تنفيذ الأوامر طالما الشرط صحيح.

الاستخدامات:

التكرار حتى يتحقق شرط معين.
بناء برامج تعتمد على الإدخال المستمر.
تنفيذ العمليات غير محددة العدد مسبقًا.

مميزات:

مرنة وتعمل حسب الشرط وليس عدد مرات محدد.
مناسبة للحالات التي لا نعرف فيها عدد التكرارات مسبقًا.

ملاحظات:

يجب تحديث الشرط داخل الحلقة لتجنب حلقة لا نهائية (Infinite Loop).
إذا أصبح الشرط False تتوقف الحلقة مباشرة.
